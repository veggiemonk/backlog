# Installation

## With Go

### Install from Source

```bash
git clone https://github.com/veggiemonk/backlog
cd backlog
go build .
```

### Install Directly

```bash
go install github.com/veggiemonk/backlog@latest
```

## Download Binary

Download the latest binary from the [releases page](https://github.com/veggiemonk/backlog/releases).
